正则练习

练习实现最常用的正则表达式，分析肯能遇到的问题及解决办法

  • 写一个验证QQ号码的正则表达式(注意：连续调用两次test()方法后不要第二次的test总会返回一个false的结果)
// 验证QQ
var regexpQQ = /^[1-9]\d{4,10}$/gi;
var str = "123456";     //返回true
alert(regexpQQ.test(str));
// 1、每个正则表达式都有一个 lastIndex 属性，用于记录上一次匹配结束的位置，下一次调用时从该位置继续搜索，解决办法，将正则表达式的 lastIndex 属性设置为0 
regexpQQ.lastIndex=0;
// 2、把g去掉就行了，g就是继续往下的意思，不是必要别加g，i是不区分大小写的，这里也没有必要，直接var regexpQQ = /^[1-9]\d{4,10}$/ 就可以
总结：一共有两种解决办法
1. 正则模式去掉全局模式
2. 重新设置lastIndex为0
